// steps to work with API

Step 1.)  Create Driver using create-driver route. Copy the driverId. Will be needed in the next step.
Step 2.)  Populate the database using random-points route or you can add single point by using add-location route
Step 3.)  Calculate the distance travelled by the driver using calculate-distance route.
Step 4.)  Get the Hotspots for a city for a particular day by using get-hotspots route. (Generated by DBSCAN Algorithm)
